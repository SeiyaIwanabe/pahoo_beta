# アプリケーション概要
ダンサーにフォーカスしたSNSマッチングアプリです。
<br>
10代~20代中盤の学生ダンサーや社会人ダンサーをメインターゲットとして、サービスを設計しました。
<br>
僕自身、大学からダンスを始めましたが、外部のつながりを作りにくいという課題を感じていました。
<br>
この課題を解決できないかと考え、アプリケーション制作に取り組みました。
